What

imginator is a simple image placeholder generator. Yes, there are a lot of them out there, but they're not mine ;). I like to do things myself, mostly for fun, but also for educational purposes. Simply put, it just generates placeholder images, but there are a few options you can pass.

How

Size

You can set the size by passing the dimensions like 300x300 or just 300 for square images. The size is always the first option to pass.

<img src="//img.static.my.id/300x300">
<img src="//img.static.my.id/300">

Color

You can set the background- & textcolor by passing hex color values like /ccc/999. First value (after the size) is the background color, the second is the text color. You can pass only the background color or both.

<img src="//img.static.my.id/300/ccc">
<img src="//img.static.my.id/300/ccc/999">

Format

You can set the image format by adding the file extension (including the .) to any of the 3 options. The supported image formats are .png, .gif, .jpg and .jpeg. If you pass no image format, .png is used.

<img src="//img.static.my.id/300.png/ccc/999">
<img src="//img.static.my.id/300/ccc.gif/999">
<img src="//img.static.my.id/300/ccc/999.jpg">
<img src="//img.static.my.id/300/ccc.jpeg/999">

Custom Text

You can set the text by passing another option like &text=custom+text. The &text= is always at the end. Spaces should be replaced by a + sign.

<img src="//img.static.my.id/300&text=custom+text">
<img src="//img.static.my.id/300/ccc&text=custom+text">
<img src="//img.static.my.id/300/ccc/999&text=custom+text">
<img src="//img.static.my.id/300/ccc.gif/999&text=custom+text">
